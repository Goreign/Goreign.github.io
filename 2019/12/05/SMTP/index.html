SMTP对象创建函数，以QQ邮箱为例
host: 服务器主机地址，懒得DNS找IP用域名就好
port: SMTP服务端口号，一般/默认为25

local: 本机即指定localhost即可

网易及腾讯邮箱为安全而使用SSH协议，收发服务器端口号为465/994

创建函数亦可为
smtpObj = smtplib.SMTP_SSL('smtp.qq.com',465,'localhost')
但465成功发送，994报错

登陆前记得连接服务器

送信函数
SMTP.sendmail(sender,reciever,msg)
注意msg
如报错
TypeError: expected string or bytes-like object
即为数据类型不匹配，使用.as_string()转换
